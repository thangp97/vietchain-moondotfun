
# moondotfun

Bài dự thi của Nhóm moondotfun về Phần thi Track 2, cuộc thi VietChain



## Tech Stack

**Client:** React

**Server:** Hardhat, Sepolia, Moralis


## Installation

Install moondotfun with npm

in back-end

```bash
  npm install 
  npm install dotenv
  npm intit -y
  npm install -save-dev-hardhat
```

in front-end
    
```bash
  npm install 
  npm install dotenv
```
